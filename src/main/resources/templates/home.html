superbowldb.com is a free JSON API for Super Bowl data and statistics. The Super Bowl is the annual championship game of the National Football League.

USAGE

Retrieve single Super Bowl Data in JSON format for 26th Super Bowl.

GET https://api.superbowldb.com/superbowl/{26}

Retrieve all Super Bowl Data.

GET https://api.superbowldb.com/superbowls

Retrieve a list of available categories.

GET https://api.superbowldb.com/catergories

Retrieve a desired catergory data for a single Super Bowl

GET https://api.superbowldb.com/{catergory}/{SB number}
